# Unity-UI-Extensions

## Variant 1
- Select "Add package from git URL"
- Paste <code>https://github.com/IvanMurzak/Unity-UI-Extensions.git#upm</code>

![alt text](https://neogeek.dev/images/creating-custom-packages-for-unity-2018.3--package-manager.png)

Modify manifest.json file.
<pre><code>{
    "dependencies": {
        "extensions.unity.ui": "https://github.com/IvanMurzak/Unity-UI-Extensions.git#upm"
    }
}
</code></pre>
